Request Token

วิธีการขอ Token จาก Line Notify

ขั้นตอนการออก Token

ขั้นตอนที่ 1 : ทำการเข้าที่เว็บไซต์ https://notify-bot.line.me/th/
เพื่อที่จะทำการขอ Token ทำการคลิก "เข้าสู่ระบบ"


ขั้นตอนที่ 2 : ทำการเข้าสู่ระบบของ Line


ขั้นตอนที่ 3 : ให้ทำการคลิกที่ชื่อแล้วเลือก "หน้าของฉัน"


ขั้นตอนที่ 4 : ให้ทำการกด ออก Token


ขั้นตอนที่ 5 : จะได้หน้าการออก Token


ขั้นตอนที่ 6 : ขั้นตอนการออก
1. ให้ทำการใส่ชื่อแจ้งเตือนเป็น Exam Notify
2. ให้ทำการเลือก "รับการแจ้งเตือนแบบตัวต่อตัวจาก Line Notify"
3. ให้ทำการออก Token


ขั้นตอนที่ 7 : ให้ทำการกด "คัดลอก"


ขั้นตอนที่ 8 : ทำการคลิกที่หน้าเว็บไซด์ "SEND TOKEN"
ทำการกรอกชื่อ(ไม่ต้องมีคำนำหน้า) และนำ Token ที่ออกมานำมาใส่และทำการกดส่ง
